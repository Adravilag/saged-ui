Por que SagedUI?

Construido con las mejores practicas para cualquier proyecto

WEB

Framework Agnostic

Funciona con React, Vue, Angular, o HTML puro. Web Components estandar.

A11Y

Accesible

Cumple con WCAG 2.1. Navegacion por teclado y soporte para lectores de pantalla.

CSS

Personalizable

CSS Custom Properties para theming. Adapta los colores a tu marca.

FAST

Ligero

Tree-shaking automatico. Solo carga los componentes que usas.

RWD

Responsive

Disenado mobile-first. Funciona perfectamente en cualquier dispositivo.

TS

TypeScript

Tipos incluidos. Autocompletado y validacion en tu IDE.

Sistema de Temas Global

Un sistema de design tokens que aplica a todos los componentes

Selecciona un tema:
Vista previa del tema data-theme=""

Como usar los temas

<!-- Importa los tokens globales -->
<link rel="stylesheet" href="saged-ui/dist/saged-ui.css">

<!-- Aplica el tema -->
<body data-theme="dark">
  <sg-article-editor></sg-article-editor>
</body>
